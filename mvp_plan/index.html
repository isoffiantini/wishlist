🚀 Wishlist Platform

Gestión Avanzada de Listas de Deseos - Site del Proyecto

ÉPICA 1: Gestión de Múltiples Wishlists

Estado: Backend Ready

Permitir a los usuarios crear y administrar múltiples listas de deseos organizadas

US-1.1: Creación de Nueva Wishlist

Como usuario registrado quiero crear una nueva wishlist con nombre y descripción para organizar productos por categorías específicas

API: POST /api/v1/wishlists
Headers: { Authorization: Bearer {token} }
Body: {
  "name": "Mi Wishlist de Cumpleaños",
  "description": "Regalos que me gustaría para mi cumpleaños",
  "is_public": false,
  "cover_color": "#FF6B6B"
}
Response: 201 Created
{
  "id": "wl_abc123",
  "name": "Mi Wishlist de Cumpleaños",
  "description": "Regalos que me gustaría para mi cumpleaños",
  "item_count": 0,
  "created_at": "2024-01-15T10:30:00Z"
}

US-1.2: Listado de Wishlists

Como usuario quiero ver todas mis wishlists en un panel principal para tener una visión general

API: GET /api/v1/wishlists
Headers: { Authorization: Bearer {token} }
Query Params: ?page=1&limit=20&sort=updated_at
Response: 200 OK
{
  "wishlists": [
    {
      "id": "wl_abc123",
      "name": "Mi Wishlist de Cumpleaños",
      "description": "Regalos que me gustaría...",
      "item_count": 5,
      "is_default": false,
      "updated_at": "2024-01-20T15:45:00Z"
    }
  ],
  "pagination": {
    "total": 8,
    "page": 1,
    "total_pages": 1
  }
}

US-1.3: Actualización de Wishlist

Como usuario quiero editar el nombre y descripción de cualquier wishlist existente para mantenerla actualizada

API: PUT /api/v1/wishlists/{wishlist_id}
Headers: { Authorization: Bearer {token} }
Body: {
  "name": "Wishlist Actualizada",
  "description": "Nueva descripción",
  "cover_color": "#4ECDC4"
}
Response: 200 OK
{
  "id": "wl_abc123",
  "name": "Wishlist Actualizada",
  "description": "Nueva descripción",
  "updated_at": "2024-01-20T16:00:00Z"
}

ÉPICA 2: Sistema de Compartición Unificado

Estado: En Desarrollo

Implementar modalidades flexibles de compartición para promover viralidad

US-2.1: Generación de Enlace Público

Como usuario quiero generar un enlace público permanente para cualquier wishlist para compartirla fácilmente

API: POST /api/v1/wishlists/{wishlist_id}/share-links
Headers: { Authorization: Bearer {token} }
Body: {
  "type": "public",
  "expires_at": null,
  "max_views": null
}
Response: 201 Created
{
  "share_id": "sh_xyz789",
  "share_url": "https://domain.com/share/shortcode123",
  "type": "public",
  "expires_at": null,
  "is_active": true
}

US-2.5: Enlace Colaborativo con Expiración

Como usuario quiero crear enlaces colaborativos con fecha de expiración para controlar el acceso temporal

API: POST /api/v1/wishlists/{wishlist_id}/share-links
Headers: { Authorization: Bearer {token} }
Body: {
  "type": "collaborative",
  "expires_at": "2024-02-15T23:59:59Z",
  "max_views": 50,
  "allow_comments": true,
  "allow_reactions": true
}
Response: 201 Created
{
  "share_id": "sh_collab456",
  "share_url": "https://domain.com/share/collab789",
  "type": "collaborative",
  "expires_at": "2024-02-15T23:59:59Z",
  "max_views": 50,
  "current_views": 0
}

US-2.7: Sistema de Comentarios Colaborativos

Como colaborador quiero agregar comentarios a productos en wishlists compartidas para dar feedback

API: POST /api/v1/shared/{share_id}/items/{item_id}/comments
Headers: { Authorization: Bearer {token} } // Opcional para usuarios registrados
Body: {
  "comment": "¡Este color es perfecto!",
  "guest_name": "María" // Solo para usuarios no registrados
}
Response: 201 Created
{
  "comment_id": "cmt_123",
  "item_id": "item_abc",
  "comment": "¡Este color es perfecto!",
  "author_name": "María",
  "created_at": "2024-01-20T14:30:00Z"
}

ÉPICA 3: Panel de Analytics de Wishlists

Estado: Planning

Proporcionar insights detallados sobre rendimiento y engagement

US-3.1: Métricas Básicas de Wishlists

Como usuario quiero ver el número total de wishlists activas vs. creadas para entender mi uso

API: GET /api/v1/analytics/wishlists/overview
Headers: { Authorization: Bearer {token} }
Query Params: ?time_range=30d // 7d, 30d, 90d, 1y
Response: 200 OK
{
  "total_created": 15,
  "active_count": 12,
  "archived_count": 3,

  "avg_items_per_list": 6.3,
  "most_popular_list": {
    "id": "wl_abc123",

  }
}

US-3.4: Métricas de Engagement por Wishlist

Como usuario quiero monitorear visualizaciones totales de mis wishlists compartidas para medir interés

API: GET /api/v1/analytics/wishlists/{wishlist_id}/engagement
Headers: { Authorization: Bearer {token} }
Query Params: ?time_range=30d&granularity=daily // daily, weekly, monthly
Response: 200 OK

US-3.7: Exportación de Reportes

Como usuario quiero exportar reportes de analytics en formato CSV/PDF para análisis externo

API: POST /api/v1/analytics/reports/export
Headers: { Authorization: Bearer {token} }
Body: {
  "report_type": "wishlist_performance", // "engagement", "conversion"
  "format": "csv", // "pdf", "json"
  "time_range": "90d",
  "wishlist_ids": ["wl_abc123", "wl_def456"]
}
Response: 202 Accepted
{
  "report_id": "rep_789",
  "status": "processing",
  "estimated_completion": "2024-01-20T15:00:00Z",
  "download_url": "/api/v1/reports/rep_789/download" // Una vez completado
}
